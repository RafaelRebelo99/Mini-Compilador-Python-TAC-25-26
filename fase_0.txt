================================================================================
SECÇÃO 1: SÍMBOLOS
================================================================================

--------------------------------------------------------------------------------
1. OPERADORES ARITMÉTICOS
--------------------------------------------------------------------------------
+       Adição
-       Subtração
*       Multiplicação
/       Divisão (resultado float)
//      Divisão inteira (floor division)
%       Módulo (resto da divisão)
**      Exponenciação

--------------------------------------------------------------------------------
2. OPERADORES RELACIONAIS
--------------------------------------------------------------------------------
==      Igual a
!=      Diferente de
<       Menor que
>       Maior que
<=      Menor ou igual a
>=      Maior ou igual a

--------------------------------------------------------------------------------
3. OPERADORES BOOLEANOS (SÍMBOLOS / BITWISE)
--------------------------------------------------------------------------------
&       AND bit a bit
|       OR bit a bit
^       XOR bit a bit
~       NOT bit a bit (complemento)
<<      Deslocamento de bits para a esquerda
>>      Deslocamento de bits para a direita

--------------------------------------------------------------------------------
4. SÍMBOLOS DE ATRIBUIÇÃO
--------------------------------------------------------------------------------
=       Atribuição simples
+=      Atribuição com adição
-=      Atribuição com subtração
*=      Atribuição com multiplicação
/=      Atribuição com divisão
//=     Atribuição com divisão inteira
%=      Atribuição com módulo
**=     Atribuição com exponenciação
&=      Atribuição com AND bitwise
|=      Atribuição com OR bitwise
^=      Atribuição com XOR bitwise
<<=     Atribuição com deslocamento à esquerda
>>=     Atribuição com deslocamento à direita
:=      Operador walrus (atribuição em expressão)

--------------------------------------------------------------------------------
5. SÍMBOLOS IDENTIFICADORES DE TIPOS DE DADOS
--------------------------------------------------------------------------------
" " / ' '   Delimitadores de strings
""" """     Delimitadores de strings multilinhas
[ ]         Identificador de lista (list)
{ }         Identificador de dicionário (dict) ou conjunto (set)
( )         Identificador de tuplo (tuple)
.           Separador decimal em floats / acesso a atributos
0-9         Dígitos - identificam literais inteiros ou floats
0x          Prefixo de literal hexadecimal (ex: 0xFF)
0o          Prefixo de literal octal (ex: 0o77)
0b          Prefixo de literal binário (ex: 0b1010)
j / J       Sufixo de número complexo (ex: 3j)

--------------------------------------------------------------------------------
6. SÍMBOLO IDENTIFICADOR DE INÍCIO DE BLOCO
--------------------------------------------------------------------------------
:           Dois pontos - marca o início de um bloco de código
            (usado após if, elif, else, for, while, def, class, try, etc.)
            O bloco é delimitado pela indentação (espaços/tab)
